Homework from Mintos

=====================

Command order:
>chmod +x setup.sh
>bash setup.sh

=====================

Enable port forwarding to check the web UI:
>kubectl port-forward sonarqube-sonarqube-0 9000:9000 -n sonarqube

=====================

Access 127.0.0.1:9000 in your browser

Default password: admin
